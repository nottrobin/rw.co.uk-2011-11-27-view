Progress updates

Big improvements

Wow it's been almost a year! How embarassing!

Though I'm still just playing around with my site, lots had happened in the last week or two:

  • Apache2 has been completely removed in favour of lighttpd - which appears to perform faster with lower memory footprint.

  • MySQL has been completely removed in favour of SQLite - once again SQLite is simpler with lower memory footprint.

  • This page is now running as a Catalyst application, which should hopefully enable me to rapidly be adding bits of functionality henceforth.

  • This page is now written in (I think) valid XHTML 5 - which isn't actually properly supported by a single browser yet, but that's okay because it's impressively backward-compatible.

Pretty exciting stuff. At leasy for me.

Look for more updates soon, I hope.
